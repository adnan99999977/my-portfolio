# Naif Adnan – Portfolio Website

This is a modern **Next.js portfolio website** showcasing projects, skills, experience, and a fully functional contact form that sends messages directly to Gmail using **Nodemailer**.

**Live Demo:** [https://your-live-portfolio-link.com](https://your-live-portfolio-link.com)  
**GitHub Repository:** [https://github.com/adnan99999977/portfolio](https://github.com/adnan99999977/portfolio)

---

##  Features

- **Responsive Design** – Built with TailwindCSS, optimized for all devices.
- **Projects Showcase** – Interactive project cards with:
  - Name
  - Tools used
  - Role
  - Description
  - GitHub and Live Demo links
- **Smooth Navigation** – Scroll smoothly to different sections.
- **Contact Form** – Send messages directly to Gmail (optionally Telegram).
- **Form Validation** – Required fields check, email validation, toast notifications.

---

##  Technologies Used

- **Frontend**: Next.js, React, Tailwind CSS, sass
- **Backend**: Node.js,
- **Email/Notification**: Nodemailer, Gmail App Password
- **Other Tools**: Cloudinary (image hosting), React Icons, React Toastify
